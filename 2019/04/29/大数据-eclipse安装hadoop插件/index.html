安装eclipse

与安装hadoop步骤相同，下载解压即可。下面记录如何设置eclipse桌面快捷方式。

这里的Exec和Icon都是eclipse安装目录下的对应文件，在填写的过程中需要注意最好不要带空格，否则会报找不到命令的错误。
如果不知道自己的文件是否有错，可以用desktop-file-validate eclipse.desktop命令来查看。
完成以后保存退出，来到桌面运行eclipse.desktop文件，点击确认，文件的图标就变成eclipse的图标了。

eclipse中设置hadoop安装目录

下载hadoop-eclipse-plugin-2.6.0.jar(https://github.com/winghc/hadoop2x-eclipse-plugin/blob/master/release/hadoop-eclipse-plugin-2.6.0.jar) ，放到eclipse/plugins目录下，重启eclipse。
菜单栏中选择“Window->preferences”，对话框中选择Hadoop Map/Reduce，然后在右边的“Hadoop installation derectory”中输入hadoop的安装目录。
